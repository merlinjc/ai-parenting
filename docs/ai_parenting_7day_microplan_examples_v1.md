# 18—48个月幼儿家长辅助型 AI 产品：7 天微计划示范样稿 V1

## 一、文档目标

本稿承接《7 天微计划模板 V1》，进一步回答一个更实际的问题：**当模板真正落到家长端时，一份完整的周计划应该长什么样。** 与模板文档不同，这里不再只描述结构，而是直接给出两份可阅读、可执行、可复盘的示范样稿，用来验证模板是否足够清楚、是否能嵌入家庭日常，以及是否保持了“非诊断化、低负担、强亲子”的产品边界。

本轮样稿刻意选取两个最常见、也最能代表家庭真实压力的高频焦点。第一类，是 **24—36 个月阶段的“选择表达”**，它对应大量“想要什么却只会拉扯、哭闹、直接抢”的日常场景。第二类，是 **24—36 个月阶段的“过渡预告”主焦点 + “情绪命名”辅助焦点**，它对应“从玩到收、从外出到回家、从自由活动到洗澡睡前”这类高冲突转场。前者验证“表达支持型计划”是否足够具体，后者验证“情绪调节型计划”是否能在安全边界内保持克制。

| 本稿用途 | 说明 |
|---|---|
| **验证模板可读性** | 检查模板落成完整周计划后，是否仍然容易理解 |
| **验证模板可执行性** | 检查每日动作是否足够短、足够具体、能嵌入家庭现实 |
| **验证跨场景适配** | 检查同一焦点能否从主场景迁移到第二场景 |
| **验证边界完整性** | 检查不适用条件、升级门槛、建议咨询口径是否清楚 |
| **为后续页面设计提供样本** | 供“本周计划页、日计划页、周反馈页、即时求助页”直接承接 |

## 二、样稿选择原则

这两份示范计划并不试图覆盖全部八类焦点，而是优先覆盖最能体现产品价值的两种计划类型。第一种，是**把表达机会从成人猜测改成儿童主动表达**。第二种，是**把高压转场从临时应付改成提前准备**。这两类场景在家庭中出现频率高，且与家长主观困扰度高度相关，最适合检验微计划是否真的能帮助家长从“着急处理”走向“稳定支持”。

| 样稿 | 代表的计划类型 | 选择理由 |
|---|---|---|
| **样稿一：选择表达** | 表达支持型计划 | 高频、低门槛、与日常吃饭拿物强相关，最能测试“等待 + 选择 + 短句模板”是否易用 |
| **样稿二：过渡预告 + 情绪命名** | 调节支持型计划 | 高频、高冲突、最能体现产品的安全边界与家长即时话术价值 |

## 三、示范样稿一：24—36个月《选择表达》周计划

这份样稿适用于这样一类家庭情境：孩子已经有明显表达意图，但在高需求场景里仍然更常通过拉手、抢东西、哭闹或反复喊叫来表达，家长需要频繁猜测其需求。这里的目标不是要求孩子一周内把表达变成完整句，而是帮助家长先把“猜—急—立刻满足”的旧节奏，改成“停顿—给选择—等孩子用动作或词语表达”的新节奏。

### 1. 计划概况

| 槽位 | 内容 |
|---|---|
| **计划标题** | 24—36个月：选择表达周计划——把“拉扯和哭闹”慢慢转成“可看见的表达” |
| **适用对象** | 24—36 个月；主主题为**表达需求与语言理解**；风险层级为**重点关注** |
| **关联观察条目** | 24B-01（开始使用双词句或短句表达需求）、24B-03（能否在两个选择中表达选择） |
| **典型家庭困扰** | 想要零食、想换玩具、想自己来时，经常先哭、拉扯或直接抢，家长很难等到明确表达 |
| **本周家长目标** | 在两个高频场景中，稳定使用“二选一 + 停顿等待”的结构，让孩子更常用动作、词语或短句表达想要什么 |
| **辅助目标** | 减少家长替孩子过快猜测和立即满足的频率 |
| **优先场景** | 点心时间、选玩具、穿衣前后 |
| **不适用条件** | 孩子生病、强烈困倦、已进入高强度哭闹且无法接收语言时；成人无法维持稳定语速时 |
| **本周成功标准** | 一周内至少在 2 个场景中出现更清楚的选择表达；家长猜测次数减少；冲突前出现表达尝试 |

### 2. 七日安排

| 天数 | 当日主练习 | 自然嵌入机会 | 示范话术 | 观察记录点 | 日终复盘问题 |
|---|---|---|---|---|---|
| **Day 1** | 在点心时间固定做一次“二选一”，只给两个真实选项，不追求开口说完整 | 晚上选睡前书时再给一次二选一 | “你要苹果还是香蕉？”；“你可以指给我看。” | 孩子是直接哭闹，还是愿意看、指或说一个词；家长是否能等 2—5 秒 | 今天最容易等到表达的是哪一刻？ |
| **Day 2** | 重复 Day 1 的点心场景，保持相同问法和相同停顿长度 | 出门前在鞋子或外套上给一次二选一 | “苹果还是香蕉？”；“你想自己拿，还是我帮你？” | 与 Day 1 比，孩子是否更快理解结构；是否仍需要家长不断重复 | 今天的问法是否比昨天更稳定？ |
| **Day 3** | 把“二选一”迁移到选玩具场景，仍只给两个选项 | 洗澡前给“先拿鸭子还是先拿杯子”的小选择 | “你要小车还是积木？”；“你选一个。” | 孩子是否只能在吃的场景表达，还是玩具场景也能接住 | 第二场景里，孩子最常用什么方式表达？ |
| **Day 4** | 在原有两个场景里，家长少说一点，把更多空位留给孩子先看、先指、先说 | 共读时让孩子在两页图里选“先看哪一页” | “你来选。”；“我等你告诉我。” | 是否出现更主动的看向大人、指向或单词；家长是否忍住提前说出答案 | 今天有没有一刻是孩子先发起，而不是大人先猜到？ |
| **Day 5** | 在孩子着急时，先命名意图，再给选择，不立刻满足 | 穿衣时给“先穿上衣还是先穿裤子” | “你很想快点拿到，现在你选苹果还是香蕉。”；“先上衣，还是先裤子？” | 在高需求情境下，孩子是否还能接住选择；冲突前是否出现表达尝试 | 今天最难的一次，是因为孩子太急，还是因为选项不够清楚？ |
| **Day 6** | 由另一位照护者在一个熟悉场景中重复同样结构 | 外出途中让孩子在两种小零食或两种活动里做选择 | “轮到你选。”；“你指给我看也可以。” | 方法是否只对一个照护者有效；不同时间段是否有差异 | 换照护者后，结构还容易被接住吗？ |
| **Day 7** | 回看本周最有效的两个场景，只保留最容易成功的表达方式 | 晚上做一次轻松的“你想先看书还是先收玩具” | “这周你越来越会告诉我你想要什么了，我们今天还像前几天那样来。” | 是否在两个场景中都出现更清楚表达；是否值得继续同焦点 1 周 | 下周最值得继续保留的一个场景是什么？ |

### 3. 周末判断与下一步

| 周末结论 | 判断依据 | 建议动作 |
|---|---|---|
| **继续同焦点** | 已开始出现更清楚的指向、看向、单词或短句；家长能稳住等待节奏 | 再巩固 1 周，但只保留最有效的 2 个场景 |
| **继续但降难度** | 孩子在平静时能表达，但一着急就回到哭闹和拉扯 | 缩减为 1 个核心场景，减少问题长度，只保留动作或单词回应 |
| **换辅助焦点** | 孩子愿意表达，但总接不住轮次或等待困难 | 在同主题下转向“等待回应”或“轮流支架” |
| **建议补充观察** | 这一周只在单一场景尝试，或照护者做法差异太大 | 先补 1 周记录，再决定是否继续 |
| **建议咨询** | 多场景持续只能依靠高强度哭闹表达，且 2 周以上无明显改善，或伴随退步 | 进入咨询建议路径，不继续单靠家庭训练加量 |

> 建议咨询口径：**近阶段记录显示，孩子在多个高需求场景中仍较难通过动作、词语或短句表达需求，家庭支持后改善有限。建议家长在继续保留日常温和支持的同时，与儿保或相关专业人员进一步沟通，以获得更准确的判断与建议。**

## 四、示范样稿二：24—36个月《过渡预告 + 情绪命名》周计划

这份样稿适用于另一类非常高频的家庭情境：孩子一听到“要洗澡了”“要收玩具了”“要出门了”就快速升级抗拒，家长往往只能在最后时刻临时催促、抱走或提高音量。这里的主焦点是**过渡预告**，辅助焦点是**情绪命名**。换句话说，本周的重点不是让孩子马上完全配合，而是帮助家长先把“突然打断”改成“提前预告”，把“只剩对抗”改成“先让孩子感觉被看见，再进入下一步”。

### 1. 计划概况

| 槽位 | 内容 |
|---|---|
| **计划标题** | 24—36个月：过渡预告周计划——把“突然被打断”慢慢变成“有准备地转场” |
| **适用对象** | 24—36 个月；主主题为**情绪过渡与共同调节**；风险层级为**重点关注** |
| **关联观察条目** | 24D-01（被拒绝、等待或停止喜欢活动时明显不高兴）、24D-03（固定高压场景的反应强度变化） |
| **主焦点 / 辅助焦点** | 主焦点为**过渡预告**；辅助焦点为**情绪命名** |
| **典型家庭困扰** | 一到收玩具、洗澡、出门或睡前转换，就容易哭闹、僵持、倒地或需要大人强行介入 |
| **本周家长目标** | 在两个固定高压场景中，用“提前预告 + 先后语言 + 简短情绪命名”降低转场的突然性 |
| **优先场景** | 结束玩耍去洗澡；结束客厅活动去睡前流程 |
| **不适用条件** | 孩子已经处于极高强度失控状态；家庭当日节奏明显紊乱；成人无法保持简短稳定表达 |
| **本周成功标准** | 至少一个高压场景里，孩子从“立刻升级”转为“虽抗拒但更快进入下一步”；身体强制介入减少 |

### 2. 七日安排

| 天数 | 当日主练习 | 自然嵌入机会 | 示范话术 | 观察记录点 | 日终复盘问题 |
|---|---|---|---|---|---|
| **Day 1** | 只选择一个最常爆发的转场场景，例如“玩完去洗澡”，提前 1 分钟预告，不讲太多道理 | 晚上结束一个小活动时也做一次简短预告 | “再玩一次，我们就去洗澡。”；“先收这一块，然后去洗澡。” | 预告后孩子是立刻升级，还是出现短暂停顿；成人是否在最后一刻才提醒 | 今天预告最早是在什么时候说出的？ |
| **Day 2** | 保持同一转场场景和同一话术结构，不临时改变规则 | 睡前关灯前做一次“先刷牙，再看书” | “先收这一块，再去洗澡。”；“我知道你还想玩。” | 与 Day 1 比，孩子对同一句式是否更熟悉；成人是否更能稳住语速 | 同样的话术，今天是否更容易接住？ |
| **Day 3** | 把同样结构迁移到第二个高压场景，例如“客厅活动转睡前流程” | 出门前也做一次短预告 | “再看这一页，我们就去刷牙。”；“先关灯，再看书。” | 是否只能在洗澡场景有效，还是第二场景也能降低突然性 | 第二场景里，孩子最卡住的是哪一步？ |
| **Day 4** | 在预告后加入一句简短情绪命名，但不长篇安抚 | 晚饭后收玩具时也用一句情绪命名 | “你还想玩，现在要停下来，会不高兴。”；“我知道你不想结束。” | 情绪命名后，孩子是否更容易停下来听下一句；成人是否忍住连续解释 | 今天哪一句话最能让孩子感觉被看见？ |
| **Day 5** | 在固定转场里加入一个可预测动作，例如“最后收一个玩具”“自己按浴室灯” | 睡前让孩子自己选一本最后要看的书 | “最后一个玩具，收好就去洗澡。”；“你来按灯，然后我们进去。” | 可预测动作是否让转场更顺；抗拒是否从立刻爆发变成可协助推进 | 哪个“小动作”最能帮助进入下一步？ |
| **Day 6** | 由另一位照护者在一个熟悉转场中重复同样结构，不额外加条件 | 外出回家时尝试一次“先脱鞋，再喝水” | “你还想在外面，我们先脱鞋，再喝水。”；“我和昨天一样提醒你。” | 方法是否依赖单一照护者；换时间段后是否仍有效 | 换照护者后，孩子最需要保留的仍是哪一步？ |
| **Day 7** | 回看本周两个转场场景，保留最有效的预告时间和一句最有效话术 | 晚间只做一次轻量复现，不额外加新要求 | “这次和前几天一样，先收最后一个，再去洗澡。”；“你不想停，我知道，我们一起做下一步。” | 是否至少一个场景里升级速度下降；是否减少抱走、拖拽或提高音量 | 下周应继续同场景巩固，还是转到新的转场场景？ |

### 3. 周末判断与下一步

| 周末结论 | 判断依据 | 建议动作 |
|---|---|---|
| **继续同焦点** | 至少一个转场场景更容易从提醒进入动作，虽仍抗拒但回稳更快 | 再巩固 1 周，只保留 1—2 个最有效的预告结构 |
| **继续但降难度** | 预告方向正确，但场景太多、要求太快，家长和孩子都吃力 | 缩减为 1 个核心转场场景，不再同时练多个 |
| **换辅助焦点** | 预告有效，但真正卡住的是孩子在受挫后无法恢复 | 保持预告为基础，辅助焦点转向“情绪命名”或“选择表达” |
| **建议补充观察** | 只在单一场景记录，无法判断是否存在场景偏置 | 先补更多场景记录，再决定是否继续同计划 |
| **建议咨询** | 多个高压转场连续两周显著升级，恢复明显变慢，或伴随能力退步与家庭困扰持续升高 | 进入咨询建议路径，并准备关键事件记录供专业人员参考 |

> 建议咨询口径：**近阶段记录显示，孩子在多个高压转场场景中持续出现较强烈的升级反应，家庭支持后缓解有限，且日常生活受到明显影响。建议家长在继续保持简短、稳定、低刺激支持的同时，与儿保或相关专业人员进一步沟通，以获得更合适的评估与建议。**

## 五、两份样稿的验证结论

从结构上看，这两份样稿已经能够验证模板 V1 的三个关键假设。第一，**同一模板可以同时承载“表达支持型计划”和“调节支持型计划”**，且不会丢失统一性。第二，**七日节奏是够用的**，它既能提供稳定骨架，又允许焦点之间替换具体动作。第三，**安全边界可以直接写进家长可读内容中**，而不需要另起一层专业文档解释。

| 验证点 | 结论 |
|---|---|
| **模板是否能写成完整周计划** | 可以，且结构清晰 |
| **每日任务是否足够短** | 可以，基本可嵌入家庭日常 |
| **两类不同焦点是否都能适配** | 可以，表达支持与转场调节都能落入同一骨架 |
| **家长话术是否保持克制** | 可以，未出现诊断化或强评判表达 |
| **升级与咨询边界是否清楚** | 可以，已能直接进入风险升级链路 |

## 六、下一步建议

在示范样稿已经可读、可执行之后，后续最自然的推进顺序会更具体。首先，可以把这两份样稿继续翻译成**页面级信息架构**，拆成“计划卡片、今日任务、记录浮层、周总结页”的信息层次。其次，可以再补一份 **18—24 个月的等待回应样稿**，验证模板在更低龄阶段是否仍然自然。最后，可以补一份 **建议咨询情境下的稳定支持样稿**，让“日常支持—重点关注—建议咨询”三种输出形成完整闭环。
